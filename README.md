# Portfolio Website

A responsive, dark-themed engineering portfolio built with Flask and Bootstrap 5. This site is designed to bridge the gap between academic research and industrial R&D, showcasing a background in Electrical and Computer Engineering.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, Custom CSS, Bootstrap 5
* **Deployment:** Render

## Features
* **Dynamic Project Navigation:** Linking from the Experience section on the home page directly to auto expanded technical details on the Projects page.
* **Technical Dashboard UI:** A high contrast, industrial aesthetic using custom badge components for job titles and skill sets as well as hover effects.
* **Responsive Design:** Fully optimized for mobile and desktop, featuring a professional sticky navbar and a photography gallery.
